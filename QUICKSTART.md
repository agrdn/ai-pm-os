## Step 2: Take Your First Note (1 min)

After your next meeting, create a new file in `Raw notes/`:

**Filename:** `2024-12-04 - Team standup.md`

**Content:** Just dump your thoughts. Example:
```
quick standup with the team
@sarah says auth component is on track for friday
@mike worried about API rate limits - might need to revisit
DEC: we're skipping the dashboard for v1, focusing on core flow
TODO: I need to update the PRD with the new scope
meeting felt good, team is aligned
```

That's it. Messy is fine.
